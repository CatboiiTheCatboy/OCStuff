# OCStuff

Something about OpenComputers, drones, robots and micromachines I suppose.

## Stuff

### DroneStuff

Some drone ROMs. Useful or not, up to your imagination.

- farm: Farms.
- remote-fly: Flies remotely.
- Etc.

### RobotStuff

Simple robot scripts.

- strip: Strip mines.
- ribs: Strip mines. Ribcage style.
- And (a bit) more.

### Robot ROMStuff

Simple robot scripts, but embedded.

- ribs: Better version of the other "ribs". Eugh...

### MicroStuff

Embed it up.

- remote-instructions: Waits for a script of Lua instructions.

### RemoteStuff

All your favorite remote programs.

- remote: Sends and receives.
- remote-1-way: Sends but doesn't receive.
- mail: Sends a massive (< 8190 B) file (but still doesn't receive).
